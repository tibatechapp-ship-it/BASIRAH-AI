# BASIRAH-AI
برنامج لبصيرة منصة ذكية لفهرسة وتصنيف وتنظيم المكتبات الرقمية والبحوث العلمية تلقائياً، تعتمد على تحليل المحتوى واكتشاف العلاقات بين الملفات وإزالة التكرار وبناء قاعدة معرفية قابلة للبحث والاستكشاف.
BASIRAH AI

بصيرة منصة معرفية ذكية تهدف إلى تحويل المكتبات الرقمية والبحوث العلمية من ملفات مبعثرة إلى قاعدة معرفية منظمة وقابلة للبحث والتحليل.

الرؤية

الانتقال من إدارة الملفات إلى إدارة المعرفة.

المميزات المخطط لها

- تصنيف المحتوى حسب الموضوع.
- اكتشاف الملفات المكررة.
- استخراج العنوان والمؤلف.
- دعم PDF و DOCX و EPUB.
- البحث الدلالي داخل المكتبة.
- بناء خرائط معرفية بين الكتب والبحوث.
- OCR للكتب المصورة.
- تطبيق سطح مكتب وتطبيق أندرويد.

خارطة الطريق

الإصدار 0.1

- قراءة PDF و TXT.
- تصنيف أولي للكتب.
- إنشاء المجلدات تلقائياً.

الإصدار 0.2

- إزالة المكرر.
- دعم DOCX.
- تقارير وإحصائيات.

الإصدار 0.3

- واجهة رسومية.
